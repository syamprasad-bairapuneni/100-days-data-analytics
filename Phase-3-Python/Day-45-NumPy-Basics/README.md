### Why NumPy?
- **100x faster** than Python lists for numerical operations
- **Vectorization** eliminates need for loops
- **Memory efficient** - fixed data types
- **Built-in analytics functions** - mean, median, std, percentile

### Core Concepts
1. **Arrays**: Foundation of NumPy - like lists but faster
2. **Broadcasting**: Operate on entire arrays at once
3. **Boolean Indexing**: Filter data based on conditions
4. **Axis operations**: Work across rows (axis=0) or columns (axis=1)
